# Landscape Illustration Research: Plant Rendering Techniques

## Executive Summary

This document compiles professional landscape architecture rendering techniques, seasonal color systems, and technical SVG implementation strategies for creating artistic plant visualizations alongside scientific representations.

**Research Date:** February 1, 2026
**Purpose:** Enable artistic rendering modes (watercolor, marker, hand-drawn) with seasonal color accuracy for the GSG Planting Plan system

## 1. Professional Rendering Styles

### 1.1 Watercolor Rendering

**Characteristics:**
- Soft, transparent washes that allow underlying layers to show through
- Loose edges with organic bleeding effects
- Color gradients from light to dark within single elements
- Paper-like texture with subtle variations
- Line-and-wash technique: ink drawing first, then watercolor overlay

**Professional Sources:**
- [Landscape Architecture Watercolor Rendering — DEPICTION](https://www.depiction.com/landscape-architecture-watercolor)
- [Watercolor Rendering for Architecture & Interior Design](https://www.myarchitectai.com/blog/watercolor-rendering)
- [James Akers Architectural Rendering - Watercolor Technique](https://www.akersarchitecturalrendering.com/blog/category/Watercolor+Technique)

**Key Techniques:**
- **Loose washes:** Large areas of transparent color
- **Line-and-wash:** Combined ink linework with watercolor fills
- **Negative painting:** Painting around shapes to define them
- **Mixed media:** Combining ink, pastel, and watercolor

**Quote from Research:**
> "Due to numerous varieties of colors and facility of combining, watercolor is considered one of the best rendering tools. This rendering style is commonly used in architecture, interior design, and landscaping to help communicate intent without locking into hard specs."

### 1.2 Marker Rendering

**Characteristics:**
- Bold, confident strokes with defined edges
- Layered color application (light to dark)
- Clean shapes with minimal blending
- Emphasis on ink drawing with proper line weights
- Marker applied FIRST, then ink overlay

**Professional Sources:**
- [Rendering Plant Symbols in Marker - Paper Garden Workshop](https://www.papergardenworkshop.com/blog/rendering-plant-symbols-in-marker)
- [Quick Tips for Rendering a Plant Symbol in Marker](https://www.papergardenworkshop.com/blog/quick-tips-for-rendering-a-plant-symbol-in-marker)
- [Render Landscape Plan with Markers - Land8](https://land8.com/forums/topic/render-landscape-plan-with-markers/)

**Recommended Materials (for reference):**
- Borden Riley 100S marker paper
- Architectural markers (Prismacolor, Chartpak, Copic)

**Quote from Research:**
> "With marker, the emphasis is placed on the ink drawing using proper line weights and hatching, while using the marker to add color below the ink, meaning you apply the marker first, then ink over it."

### 1.3 Hand-Drawn Sketching

**Characteristics:**
- Loose, organic linework
- Hatching and cross-hatching for texture
- Variable line weights
- Imperfect circles and irregular edges
- Stippling for depth and shadow

**Professional Sources:**
- [How To Draw Trees In Plan - DRAFTSCAPES](https://draftscapes.com/how-to-draw-trees-in-plan-1/)
- [Drawing Plant Symbols Practice Sheet - Paper Garden Workshop](https://www.papergardenworkshop.com/blog/drawing-plant-symbols-practice-sheet)

**Hatching Techniques:**
- **Parallel lines:** Closer together = darker value
- **Cross-hatching:** Multiple directions for depth
- **Stippling:** Dots in varying concentrations
- **Contour hatching:** Following the form's curves

**Quote from Research:**
> "Hatching is the process of making parallel lines in different concentrations, closer together or farther apart, in order to create a shading or three-dimensional effect on plant drawings. Stippling is a common rendering technique for trees in which tiny dots in various concentrations are used to form shade and three-dimensionality."

## 4. Plan View Representation Standards

### 4.1 Canopy vs. Spread Representation

**Research Sources:**
- [ENH1195/EP456: Landscape Design: Drawing a Planting Plan](https://edis.ifas.ufl.edu/publication/EP456)
- [How To Draw Trees In Plan - DRAFTSCAPES](https://draftscapes.com/how-to-draw-trees-in-plan-1/)
- [Tree Canopy - Bower & Branch](https://bowerandbranch.com/products/tree-canopy-signature-landscape-design)

**Key Findings:**

1. **Canopy = Spread = Dripline**
   - In plan view, these are essentially the same measurement
   - Represents the outermost extent of the tree's crown
   - Drawn as circles scaled to mature plant size

2. **Drawing Standards:**
   - Plants drawn as circles matching mature spread diameter
   - Typically at 1:10 scale (1 inch = 10 feet)
   - Circles represent approximate area covered by canopy

3. **Differentiation Methods:**
   - **Dense Canopy (solid):** Filled circles or dense hatching
   - **Spread/Dripline (extent):** Dashed or dotted outline
   - **Visibility Through Tree:** Small triangles or polygons inside canopy

**Quote from Research:**
> "Trees in landscape plans are typically drawn as circles with diameters matching the mature plant's spread at the plan's scale. The dripline is defined as the outermost circumference of the tree's canopy, from which water drips onto the ground."

### 4.2 Drawing Techniques for Organic Edges

**Contour Style:**
- Draw only outside edges of canopy
- Avoid template uniformity
- Move pen inside/outside guidelines
- Return to original template periodically
- Creates natural, irregular appearance

**Adding Texture:**
- Small triangles/polygons for depth
- Sharper edges for toothy/sharp leaf margins
- Stippling in varying concentrations
- Hatching following canopy curves

## 6. Deciduous vs. Evergreen Seasonal Behavior

### 6.1 Deciduous Trees

**Characteristics:**
- Lose all leaves at end of growing season
- Dramatic autumn color displays
- Bare branches in winter
- Fresh new growth in spring

**Seasonal Progression:**
- **Spring:** Light yellow-greens, translucent new leaves
- **Summer:** Deep rich greens, full canopy
- **Autumn:** Yellows → Oranges → Reds → Browns (sequential or simultaneous)
- **Winter:** Bare branches, bark texture visible

**Timing:**
- Fall color: Late September (north) to early November (south)
- Leaf drop: October - November
- Dormancy: November - March
- Bud break: March - May

**Research Sources:**
- [Understanding How Trees Change Throughout Seasons - Tallow Tree Services](https://www.tallowtreeservices.com/understanding-how-trees-change-throughout-the-seasons)
- [Science of Fall Colors - US Forest Service](https://www.fs.usda.gov/visit/fall-colors/science-of-fall-colors)

### 6.2 Evergreen Trees

**Characteristics:**
- Retain foliage year-round
- Lose needles/leaves gradually, not all at once
- Some seasonal color variation
- Structural green mass in winter

**Seasonal Progression:**
- **Spring:** Slight brightening, new growth lighter green
- **Summer:** Standard deep green
- **Autumn:** Minimal change, possible slight bronzing
- **Winter:** Winter bronzing in some species, older needle drop

**Seasonal Needle Drop:**
- Occurs primarily in fall
- Older needles (toward center of tree) drop
- Red pine: deep copper color before falling
- White pine: golden color before falling

**Winter Bronzing:**
- Natural response to cold, wind, sunlight
- Colors: Bronze, rust, yellow-brown
- Temporary; returns to green in spring
- More pronounced in certain species (arborvitae, juniper)

**Research Sources:**
- [Seasonal Evergreen Foliage Changes - For The Love Of Trees](https://www.fortheloveoftrees.com/seasonal-evergreen-foliage-changes/)
- [Don't Panic! Your Evergreens Are Just Bronzing - Bower & Branch](https://bowerandbranch.com/blogs/all-about-trees/evergreen-bronzing)

### 6.3 Deciduous Conifers

**Special Case:** Tamarack, Larch, Dawn Redwood

- Behave like deciduous broadleaf trees
- Entire tree turns yellow, rust, brown in fall
- All needles drop
- Completely bare in winter

**Quote from Research:**
> "As we advance into fall, an even more eye-catching color change will be on deciduous conifers, most notably tamarack or larch. Entire trees turn yellow, rust, and brown since these trees shed all their needles in fall and are needleless in winter."

## 7. SVG Technical Implementation

### 7.1 feTurbulence - Perlin Noise Generation

**Purpose:** Creates organic, natural textures

**Technical Resources:**
- [SVG Filter Effects: Creating Texture with feTurbulence - Codrops](https://tympanus.net/codrops/2019/02/19/svg-filter-effects-creating-texture-with-feturbulence/)
- [MDN: feTurbulence](https://developer.mozilla.org/en-US/docs/Web/SVG/Reference/Element/feTurbulence)
- [Observable: SVG Filters - feTurbulence](https://observablehq.com/@pstuffa/svg-filters-feturbulence)

**Key Attributes:**

```svg
<feTurbulence
  type="fractalNoise"        <!-- or "turbulence" -->
  baseFrequency="0.02"       <!-- Scale of noise (lower = larger features) -->
  numOctaves="2"             <!-- Detail level (1-4 typical) -->
  seed="0"                   <!-- Random seed for variation -->
/>
```

**Type Parameter:**
- **fractalNoise:** Smoother, cloud-like (best for watercolor)
- **turbulence:** More chaotic, marbled (best for organic edges)

**baseFrequency Guidelines:**
- 0.001 - 0.01: Large, smooth variations (clouds, soft washes)
- 0.02 - 0.05: Medium detail (watercolor texture)
- 0.1 - 0.5: Fine detail (stippling, grain)

**numOctaves Guidelines:**
- 1: Simple, smooth noise
- 2-3: Natural, balanced detail
- 4+: Very detailed, can be slow to render

**Applications:**
- Watercolor paper texture
- Soft edges and bleeding
- Organic foliage variation
- Wood grain, marble, clouds

**Quote from Research:**
> "The feTurbulence SVG filter primitive creates an image using the Perlin turbulence function and allows the synthesis of artificial textures like clouds or marble."

### 7.2 feDisplacementMap - Organic Edge Distortion

**Purpose:** Warps shapes for natural, hand-drawn appearance

**Technical Resources:**
- [SVG Filter Effects: feDisplacementMap - Codrops](https://tympanus.net/codrops/2019/02/12/svg-filter-effects-conforming-text-to-surface-texture-with-fedisplacementmap/)
- [A Deep Dive Into SVG Displacement Filtering - Smashing Magazine](https://www.smashingmagazine.com/2021/09/deep-dive-wonderful-world-svg-displacement-filtering/)
- [Creating Organic Textures - DEV Community](https://dev.to/hexshift/creating-organic-textures-with-svg-filter-distortions-1moj)
- [MDN: feDisplacementMap](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/feDisplacementMap)

**Key Attributes:**

```svg
<feDisplacementMap
  in="SourceGraphic"         <!-- Input to displace -->
  in2="turbulenceResult"     <!-- Displacement map source -->
  scale="20"                 <!-- Displacement intensity -->
  xChannelSelector="R"       <!-- Red channel for X displacement -->
  yChannelSelector="G"       <!-- Green channel for Y displacement -->
/>
```

**scale Parameter:**
- 0-10: Subtle warping (slight hand-drawn feel)
- 10-30: Medium distortion (watercolor edges)
- 30-50: Strong distortion (organic, irregular)
- 50+: Extreme distortion (abstract, highly organic)

**Workflow:**
1. Generate Perlin noise with feTurbulence
2. Optionally blur the noise with feGaussianBlur
3. Apply displacement to shape using feDisplacementMap

**Example - Organic Plant Edge:**

```svg
<filter id="organicEdge">
  <feTurbulence type="turbulence" baseFrequency="0.02" numOctaves="2" result="turbulence" />
  <feDisplacementMap in="SourceGraphic" in2="turbulence" scale="15" xChannelSelector="R" yChannelSelector="G" />
</filter>
```

**Quote from Research:**
> "SVG filters can produce rich, organic distortions using primitives like feTurbulence and feDisplacementMap to generate textures that feel dynamic, analog, and alive."

### 7.3 feGaussianBlur - Soft Watercolor Effects

**Purpose:** Creates soft edges, bleeding, and transparency effects

**Key Attributes:**

```svg
<feGaussianBlur
  in="SourceGraphic"
  stdDeviation="2"           <!-- Blur radius -->
/>
```

**stdDeviation Guidelines:**
- 0.5-1: Slight softening (anti-aliasing)
- 2-5: Watercolor soft edges
- 5-10: Strong blur, bleeding effect
- 10+: Very soft, diffused

**Applications:**
- Watercolor edge bleeding
- Shadow softening
- Depth of field
- Glow effects

### 7.4 feColorMatrix - Color Adjustments

**Purpose:** Color transformation and manipulation

**Type Options:**
- **saturate:** Adjust color saturation
- **hueRotate:** Shift hues
- **luminanceToAlpha:** Convert brightness to transparency

**Example - Desaturation for Texture:**

```svg
<feColorMatrix type="saturate" values="0" />
```

### 7.5 feComposite & feBlend - Layer Combination

**Purpose:** Combine multiple filter results

**feComposite operators:**
- **over:** Standard layering
- **in:** Use intersection
- **out:** Use difference
- **atop:** Composite within bounds

**feBlend modes:**
- **normal:** Standard blend
- **multiply:** Darken
- **screen:** Lighten
- **overlay:** Contrast blend

### 7.6 feMorphology - Stippling and Texture

**Purpose:** Expand or contract shapes

**Operations:**
- **dilate:** Expand shapes (thicken)
- **erode:** Contract shapes (thin)

**Applications:**
- Stippling dot patterns
- Outline creation
- Texture generation

## 8. Complete SVG Filter Examples

### 8.1 Watercolor Plant Rendering

```svg
<defs>
  <filter id="watercolor" x="-50%" y="-50%" width="200%" height="200%">
    <!-- Paper texture -->
    <feTurbulence type="fractalNoise" baseFrequency="0.04" numOctaves="3" result="noise" />
    <feColorMatrix in="noise" type="saturate" values="0" result="desaturatedNoise" />
    <feComponentTransfer in="desaturatedNoise" result="texture">
      <feFuncA type="discrete" tableValues="0 0 0 1 1" />
    </feComponentTransfer>

    <!-- Soft edges -->
    <feTurbulence type="turbulence" baseFrequency="0.02" numOctaves="2" result="edgeNoise" />
    <feDisplacementMap in="SourceGraphic" in2="edgeNoise" scale="8" result="displaced" />
    <feGaussianBlur in="displaced" stdDeviation="2" result="softEdges" />

    <!-- Combine texture and soft edges -->
    <feBlend in="softEdges" in2="texture" mode="multiply" result="textured" />

    <!-- Subtle color variation -->
    <feTurbulence type="fractalNoise" baseFrequency="0.01" numOctaves="2" result="colorNoise" />
    <feColorMatrix in="colorNoise" type="hueRotate" values="30" result="colorVariation" />
    <feBlend in="textured" in2="colorVariation" mode="overlay" opacity="0.3" />
  </filter>
</defs>

<circle cx="100" cy="100" r="80" fill="#228B22" filter="url(#watercolor)" />
```

**Characteristics:**
- Soft, organic edges
- Paper-like texture
- Subtle color variation
- Transparent, layered appearance

## 11. Procedural Variation Techniques

### 11.1 Non-Uniform Edges

**Technique: Random Seed Variation**

```typescript
function generateUniqueFilter(plantId: string, style: 'watercolor' | 'marker' | 'sketch'): string {
  const seed = hashCode(plantId) % 1000;

  return `
    <filter id="plant-${plantId}-${style}">
      <feTurbulence seed="${seed}" ... />
      ...
    </filter>
  `;
}

function hashCode(str: string): number {
  let hash = 0;
  for (let i = 0; i < str.length; i++) {
    const char = str.charCodeAt(i);
    hash = ((hash << 5) - hash) + char;
    hash = hash & hash;
  }
  return Math.abs(hash);
}
```

## 13. Key Questions Answered

### Q1: How do professional landscape architects differentiate between canopy (dense foliage) and spread (extent) in drawings?

**Answer:**
- **Canopy (dense foliage):** Solid filled circle or dense hatching, represents actual leafy mass
- **Spread/Dripline (extent):** Dashed or dotted outline circle, represents outermost reach
- **Both** are typically the same diameter at maturity
- **Visual differentiation:** Solid interior = canopy, dashed perimeter = spread boundary

### Q2: What SVG techniques best replicate hand-drawn watercolor effects?

**Answer:**
- **feTurbulence** (fractalNoise) for paper texture and soft variations
- **feDisplacementMap** with low scale (5-10) for soft edge bleeding
- **feGaussianBlur** (stdDeviation 2-5) for transparency and soft edges
- **feColorMatrix** for subtle hue variations
- **feBlend** (multiply, overlay) for layered transparency
- **Low opacity** (0.6-0.8) for transparent wash appearance

### Q3: How can we programmatically generate organic, non-uniform edges for plant symbols?

**Answer:**
- **Seed-based variation:** Use plant ID hash as feTurbulence seed for unique randomness
- **Multiple displacement layers:** Stack 2-3 feDisplacementMap filters with different scales
- **Variable baseFrequency:** Randomize between 0.02-0.05 for edge variation
- **Asymmetric distortion:** Use different xChannelSelector and yChannelSelector values
- **Layered circles:** Overlay multiple circles with slight offsets and varying opacities

### Q4: What are the standard color conventions for seasonal plant representation?

**Answer:**

| Season | Deciduous Colors | Evergreen Colors | Palette Range |
|--------|------------------|------------------|---------------|
| **Spring** | Light yellow-greens (#C9DC87, #7CBB5D, #5B9E48) | Slightly brighter greens (#00FF7F, #89FFC4) | Yellow-greens, pastels |
| **Summer** | Deep rich greens (#228B22, #2E6F40, #155724) | Standard greens (#228B22, #4E8F38) | Deep mature greens |
| **Autumn** | Yellows to reds (#FCB829, #FE7126, #D00127, #644A2A) | Same greens, possible bronzing (#CD7F32) | Warm yellows/oranges/reds |
| **Winter** | Bare branches (browns: #BBA89A, #875D4A, #644A2A) | Deep greens with bronzing (#05472A, #637E68) | Browns, grays, deep greens |

### Q5: How do we reference real plant images (e.g., from Crocus) to extract accurate seasonal colors?

**Answer:**

**Method 1: Manual Color Extraction**
- Download seasonal images from Crocus API
- Use image processing to extract dominant colors
- Create palette of 5-10 most prominent colors
- Use for gradient fills in SVG

**Method 2: Average Color Sampling**
```typescript
async function extractDominantColors(imageUrl: string): Promise<string[]> {
  // Load image
  const img = await loadImage(imageUrl);

  // Use color quantization algorithm (e.g., median cut)
  const palette = extractPalette(img, 5);

  // Convert to hex codes

```

**Method 3: Fallback to Palette Defaults**
- If no seasonal image available
- Use plant type (deciduous/evergreen) + season
- Apply standard palette from research
